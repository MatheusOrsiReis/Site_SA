Logica Computacional


O que são fluxogramas? Qual a sua simbologia básica?


Fluxogramas são representações gráficas de processos ou algoritmos, mostrando de forma visual o passo a passo necessário para realizar uma tarefa. Eles utilizam símbolos padronizados que facilitam a compreensão do fluxo das ações. Entre os principais símbolos, estão a elipse, que indica o início e o fim do processo; o retângulo, que representa uma ação ou operação; o losango, que mostra uma decisão ou condição; o paralelogramo, usado para indicar entrada e saída de dados; e as setas, que direcionam o caminho que o fluxo deve seguir.


O que são algoritmos? Onde são usados?


Algoritmos são sequências de instruções organizadas de forma lógica e ordenada, criadas para resolver um problema ou executar uma tarefa específica. Eles estão presentes em praticamente todas as áreas da tecnologia, sendo a base de funcionamento dos programas de computador, aplicativos, sites, sistemas automatizados e até de dispositivos inteligentes.


O que é o Portugol? O que é o Portugol Studio?


O Portugol é uma linguagem de programação em português que tem como objetivo facilitar o aprendizado da lógica de programação. Ela permite que o estudante entenda a estrutura e o funcionamento de um programa sem precisar dominar uma linguagem complexa. Já o Portugol Studio é um software desenvolvido para escrever, testar e executar algoritmos feitos em Portugol. Ele ajuda o aluno a visualizar o funcionamento do código e identificar possíveis erros de forma prática e educativa.


O que são variáveis e constantes?


Variáveis são espaços de memória utilizados para armazenar valores que podem ser alterados durante a execução de um programa, como idade, soma ou nota. Já as constantes são valores que permanecem fixos e não podem ser modificados, sendo usadas para representar informações que não mudam, como o valor de π ou uma taxa fixa.


Quais os tipos de dados mais usados em algoritmos


Em algoritmos escritos em Portugol, existem diferentes tipos de dados usados para armazenar informações de acordo com suas características. Os tipos booleanos representam valores lógicos como verdadeiro ou falso. O tipo caractere armazena um único símbolo, enquanto o tipo cadeia, também conhecido como string, guarda um conjunto de caracteres, como nomes ou frases. O tipo inteiro é usado para números sem casas decimais, e o tipo real para números que possuem parte decimal. Já o vetor é uma estrutura que armazena vários valores de um mesmo tipo, organizados em posições dentro de uma sequência.
